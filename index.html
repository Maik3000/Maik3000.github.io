Projects

Bank Fraud Detection in GCP
Implementation of a real-time financial fraud detection system using a Lambda architecture on Google Cloud Platform. Pub/Sub, Dataflow, BigQuery, and BigQuery ML were integrated for data processing, automated model training, and visualization in Looker Studio.
Detail
Tennis Web App with Match Predictor.
This project uses data science techniques to predict the winner of a professional tennis match between two ATP Tour players. It uses a modular pipeline in Kedro and a model trained with XGBoost. Flask acts as an intermediate server (REST API) that connects the prediction model to the web interface.
Detail
Musical Analysis with Spark
Large scale data analysis using PySpark on a 600,000 song Spotify dataset. Data cleaning, exploratory analysis, predictive popularity modeling with Random Forest, and development of a music recommendation system based on clustering and collaborative filtering were performed.
Detail
Euro 2024 Winner Prediction
Quantitative data analysis was used to estimate the probabilities of winning Euro 2024, using features like market value and international experience of the teams. Clustering models (K-Means, DBSCAN, MeanShift) were applied to group countries and generate relative probabilities. The results were visualized in Tableau.
Detail
Auto Sales ETL Pipeline in GCP
I designed and implemented an ETL pipeline on Google Cloud Platform to process and visualize automotive sales data. The workflow ranged from CSV files stored in Cloud Storage, through transformations in Dataproc with PySpark, to analysis in BigQuery and visualization in Looker Studio. I also explored the use of Cloud Data Fusion as a low-code alternative for data processing.
